04 — Installation & setup
1
Clone & install dependencies
Requires Python 3.10+ and Windows. Create a virtual environment and install everything from requirements.txt.
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
2
Create your .env file
Copy the example and fill in your values. The .env file is never committed to git.
copy .env.example .env
# PTT keys — run detect_key.py to find the name of any key
PTT_KEY=f15
PTT_KEY_OPTIMIZE=f16

# Transcription language (leave empty for auto-detect)
WHISPER_LANG=es

# Prompt optimizer provider: anthropic | openai | gemini | ollama
OPTIMIZE_PROVIDER=anthropic

# Only fill in the key for your chosen provider
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=
GEMINI_API_KEY=
OLLAMA_API_KEY=ollama
3
Find your PTT key names
Not sure what name pynput gives to a key on your keyboard, mouse, or Stream Deck? Run detect_key.py, press the key, and copy the value it shows.
python .\detect_key.py
Press any key to see its name (Esc to quit)...
Key pressed:  <Key.f15: 0>
Use in PTT_KEY:  f15
Tip: High function keys (f13–f24) are ideal — almost no app uses them. Many gaming mice, macro keyboards, and Stream Deck devices emit them even though they're not printed on standard keyboards.
4
Optional: set up Ollama for fully local prompt optimization
Skip this step if you're using a cloud provider. If you want zero cost, zero data transmission, install Ollama and pull a model.
# Install Ollama
irm https://ollama.com/install.ps1 | iex

# Pull a model (choose one)
ollama pull phi4-mini      # recommended — best quality/speed
ollama pull llama3.2:3b    # fastest, lowest RAM (~3 GB)
ollama pull qwen2.5:7b     # best quality on capable hardware
ModelSizeRAMBest for
phi4-mini3.8B~4 GBBest quality/speed balance
llama3.2:3b3B~3 GBFastest, lowest memory
qwen2.5:7b7B~6 GBBest quality on capable hardware
Language note: Local models only reliably support English and Spanish. If WHISPER_LANG is set to any other language, the optimizer will fall back to English output. 
# Set in .env to use Ollama
OPTIMIZE_PROVIDER=ollama
OLLAMA_API_KEY=ollama   # any non-empty value
5
Run
Start the app. On first run, Whisper may download the selected model. The console confirms which provider is active.
python .\voice_typer.py
Loading Whisper model on CPU...
Voice Typer ready. PTT=dictate | PTT_OPTIMIZE=dictate+optimize [anthropic] (OK).
Press Ctrl+C or use tray Quit to exit.
Windows Startup: To launch automatically on login, run python .\install_startup.py. Remove with python .\uninstall_startup.py.
